# Portfolio (Vite + React)

This is a direct conversion of your provided HTML/CSS/JS into a **Vite + React** project.
It preserves layout, design and behavior without changing the original styling.

How to run:
1. Install dependencies: `npm install` or `yarn`
2. Start dev server: `npm run dev`
3. Build: `npm run build`

Notes:
- Fonts, Typed.js and ScrollReveal are loaded from CDN in `index.html`.
- Assets folder contains placeholder files; replace with your actual resume/CV and images.
